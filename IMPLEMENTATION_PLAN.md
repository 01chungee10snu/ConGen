# 교육 영상 자동 생성 프레임워크 - 구현 계획서

## 프로젝트 개요

**프로젝트명:** ConGen (Content Generator)  
**목표:** 사용자의 자연어 프롬프트를 입력받아 교육용 영상을 자동으로 생성하는 AI 에이전트 시스템  
**작성일:** 2025-12-12  

---

## 1. 핵심 요구사항 분석

### 1.1 기능적 요구사항 (Functional Requirements)

| ID | 요구사항 | 설명 | 우선순위 |
|:---|:---|:---|:---|
| FR-01 | 자연어 프롬프트 입력 | 사용자가 자연어로 교육 영상 요청을 입력할 수 있어야 함 | P0 (필수) |
| FR-02 | 자동 스크립트 생성 | LLM이 자연어를 분석하여 구조화된 스크립트(JSON/YAML) 생성 | P0 (필수) |
| FR-03 | 대화형 스크립트 수정 | 사용자가 생성된 스크립트를 대화로 수정할 수 있어야 함 | P1 (중요) |
| FR-04 | 이미지 생성 | 장면별 교육용 이미지/슬라이드 자동 생성 | P0 (필수) |
| FR-05 | 비디오 생성 | 이미지를 기반으로 동영상 생성 (I2V) | P0 (필수) |
| FR-06 | 음성 합성 (TTS) | 내레이션 텍스트를 자연스러운 음성으로 변환 | P0 (필수) |
| FR-07 | 자막 생성 | TTS 결과에서 타임스탬프 추출하여 자막 파일 생성 | P1 (중요) |
| FR-08 | 다국어 지원 | 번역 및 다국어 자막/더빙 생성 | P2 (부가) |
| FR-09 | 영상 조립 | 모든 자산을 하나의 최종 영상으로 병합 | P0 (필수) |
| FR-10 | 일관성 유지 | 캐릭터/스타일이 전체 영상에서 일관되게 유지 | P1 (중요) |

### 1.2 비기능적 요구사항 (Non-Functional Requirements)

| ID | 요구사항 | 설명 | 목표 수치 |
|:---|:---|:---|:---|
| NFR-01 | 응답성 | 스크립트 생성은 30초 이내 완료 | < 30초 |
| NFR-02 | 처리 시간 | 10분 영상 생성은 30분 이내 완료 | < 30분 |
| NFR-03 | 확장성 | 동시 다중 요청 처리 가능 | 10+ 동시 요청 |
| NFR-04 | 영상 품질 | 최소 1080p 해상도 지원 | 1080p/4K |
| NFR-05 | 비용 효율성 | 10분 영상 생성 비용 최적화 | < $10/영상 |

### 6.3 핵심 완화 전략

1. **Cost Control First**: Veo 사용을 최소화하는 **하이브리드 렌더링 전략** 기본 채택
   - 정적 슬라이드: 이미지 → FFmpeg Ken Burns 효과
   - 동적 장면: Veo 3.1 Fast 최소 사용
   
2. **SDK Migration**: Day 1부터 `google-genai` SDK만 사용 (레거시 코드 없음)

3. **Graceful Degradation**: 각 에이전트별 폴백 옵션 구현
   - Veo 실패 → FFmpeg 슬라이드쇼
   - Nano Banana 실패 → Imagen 3 또는 Gemini 2.0 Flash Image
   - Cloud TTS 실패 → Edge TTS (로컬)
